# Focus Reader for Claude

A Chrome extension that improves readability on Claude.ai with customizable fonts, spacing, and bionic reading.

Built this for myself because my brain was fried and I couldn't focus. Maybe it'll help you too.

## Features

**Typography Controls**
- Font family (including dyslexia-friendly options like Atkinson Hyperlegible and Lexend)
- Line height
- Letter spacing
- Word spacing

**Bionic Reading**
- Bolds the first portion of each word to anchor your eyes
- Adjustable intensity and weight
- Set to 0% to disable and just use typography settings

## Installation

1. **Download** this repository (Code → Download ZIP) and unzip it

2. **Open Chrome** and go to `chrome://extensions`

3. **Enable Developer Mode** (toggle in the top-right corner)

4. **Click "Load unpacked"** and select the unzipped folder

5. **Visit [claude.ai](https://claude.ai)** — click the extension icon in your toolbar to customize

### Other Chromium Browsers

- **Brave**: `brave://extensions`
- **Edge**: `edge://extensions`
- **Arc**: Settings → Extensions

Same steps apply.

## Usage

Click the extension icon while on claude.ai to open the settings popup:

1. Adjust typography settings (font, line height, spacing)
2. Adjust bionic reading settings (bold intensity and weight)
3. Click "Apply & Refresh Page"

Settings are saved automatically and persist across sessions.

## Available Fonts

| Font | Best For |
|------|----------|
| Atkinson Hyperlegible | Low vision, dyslexia |
| Lexend | Reading fluency, dyslexia |
| Inter | Clean, modern readability |
| Open Sans | General readability |
| Source Sans | Long-form reading |
| JetBrains Mono | If you prefer monospace |
